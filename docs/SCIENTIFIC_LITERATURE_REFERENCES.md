## 1. Primary ESA Documentation

### 1.1 Sentinel-1 Mission Documents
- **ESA-EOPG-CSCOP-TN-0014**: "Sentinel-1 Product Definition", Issue 1, Rev 2, European Space Agency, 2021.
  - *Primary reference for SLC data format, annotation structure, and calibration*
  - *Used in*: `annotation.rs`, `slc_reader.rs`, `calibrate.rs`

- **ESA-EOPG-CSCOP-TN-0007**: "Sentinel-1 Product Specification", Issue 3, Rev 7, European Space Agency, 2021.
  - *Defines radiometric and geometric accuracy requirements*
  - *Used in*: `validation.rs`, `quality_assessment.rs`

- **ESA-EOPG-CSCOP-TN-0006**: "Sentinel-1 Orbit Product Format Specification", Issue 1, Rev 4, European Space Agency, 2021.
  - *Orbital state vector format and interpolation requirements*
  - *Used in*: `terrain_correction.rs` (orbit interpolation functions)

### 1.2 TOPSAR Processing Documents
- **ESA-EOPG-CSCOP-TN-0013**: "TOPSAR Processing Chains for Sentinel-1", Issue 2, Rev 1, European Space Agency, 2020.
  - *TOPSAR burst mode processing algorithms*
  - *Used in*: `topsar_merge.rs`, `iw_merge.rs`

- **ESA-EOPG-CSCOP-TN-0010**: "Sentinel-1 Antenna Pattern Correction", Issue 1, Rev 3, European Space Agency, 2021.
  - *Azimuth antenna pattern correction for TOPSAR mode*
  - *Used in*: `topsar_merge.rs` (antenna pattern functions)

## 2. Foundational SAR Processing Literature

### 2.1 Core SAR Theory
- **Cumming, I. G., & Wong, F. H.** (2005). *Digital Processing of Synthetic Aperture Radar Data: Algorithms and Implementation*. Artech House.
  - *Chapter 4: SAR Signal Processing* → Range-Doppler processing algorithms
  - *Chapter 6: SAR Geocoding* → Coordinate transformation mathematics
  - *Used in*: `terrain_correction.rs` (range-Doppler geocoding)

- **Soumekh, M.** (1999). *Synthetic Aperture Radar Signal Processing with MATLAB Algorithms*. John Wiley & Sons.
  - *Chapter 7: Geometric Corrections* → Terrain correction algorithms
  - *Chapter 5: Range-Doppler Processing* → Fundamental SAR focusing
  - *Used in*: `terrain_correction.rs`, `core/lib.rs`

### 2.2 Speckle Filtering
- **Lee, J. S.** (1981). "Speckle analysis and smoothing of synthetic aperture radar images." *Computer Graphics and Image Processing*, 17(1), 24-32.
  - *Original Lee filter formulation*
  - *Used in*: `speckle_filter.rs` (Lee filter implementation)

- **Lopes, A., Nezry, E., Touzi, R., & Laur, H.** (1993). "Structure detection and statistical adaptive speckle filtering in SAR images." *International Journal of Remote Sensing*, 14(9), 1735-1758.
  - *Gamma MAP filter and adaptive filtering*
  - *Used in*: `speckle_filter.rs` (Gamma MAP filter)

### 2.3 Multilooking Theory
- **Oliver, C., & Quegan, S.** (2004). *Understanding Synthetic Aperture Radar Images*. SciTech Publishing.
  - *Chapter 4: Speckle Statistics* → Multilooking mathematics
  - *Chapter 5: Speckle Filtering* → ENL calculation
  - *Used in*: `multilook.rs`
